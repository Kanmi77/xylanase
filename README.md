# xylanase

Project structure for acquisition, curation, annotation, and homology modeling of bacterial and fungal GH10/GH11 xylanases.

## Data sources
- UniProt
- NCBI RefSeq
- CAZy

## Annotation targets
- Organism type: bacterial/fungal
- GH family: GH10/GH11
- Optimum temperature
- Optimum pH
- Available 3D structure via PDB

## Structure generation
- SWISS-MODEL for sequences lacking structure

## Reproducibility
All commands and processing steps are stored in `scripts/`.
